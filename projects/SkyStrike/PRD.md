# PRD — SkyStrike: Retro Aerial Assault

## 1. Vision
Создать захватывающий веб-шутер (Shoot 'em up) с вертикальным скроллингом, вдохновленный классикой вроде 1942 и Sky Force. Игра должна работать плавно в браузере и иметь современную графику на базе HTML5 Canvas.

## 2. Key Features
- **Player:** Самолет с плавным управлением (клавиатура WASD/Стрелки + Пробел для стрельбы).
- **Enemies (6 видов):**
  1. *Scout:* Быстрый, слабая броня.
  2. *Interceptor:* Стреляет очередями.
  3. *Bomber:* Медленный, много здоровья, мощные снаряды.
  4. *Kamikaze:* Пытается протаранить игрока.
  5. *Shield-ship:* Защищен спереди.
  6. *Elite:* Маневренный враг.
- **Boss:** Огромный крейсер с несколькими фазами (турели, двигатели, ядро).
- **Bonuses:** Double Fire, Missile Strike, Shield, Time Warp.

## 3. Tech Stack
- HTML5 Canvas API.
- Vanilla JavaScript (модульный подход).
- CSS3 для UI-слоев (меню, HUD).

## 4. Технические ограничения (КРИТИЧНО)
- **Aspect Ratio:** Игра должна иметь фиксированное вертикальное соотношение сторон (9:16), например, логическое разрешение 1080x1920. Canvas должен центрироваться по середине экрана (Letterboxing), чтобы избежать растягивания на широких мониторах.
- **Entity Integrity:** Жесткое разделение сущностей по типам (`player`, `enemy`, `player_projectile`, `enemy_projectile`). Запрещено наносить урон самому себе.
- **Memory Management:** Все объекты, выходящие за пределы экрана, должны помечаться на удаление (`markedForDeletion`), чтобы избежать утечек памяти.
- **Performance:** Стабильные 60 FPS, использование `requestAnimationFrame` с ограничением `deltaTime` (защита от лагов при переключении вкладок).
