## Tasks
- [x] {{TASK:5.1}} Система усилителей (Power-ups). `[Role: Game Gameplay Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/PowerUps.js`. Обновить `WaveDirector.js`, `Player.js` и `CollisionManager.js`.
  - **Как сделать:** Создать базовый класс `PowerUp` (`type = 'powerup'`) и три наследника:
    1. **ShieldPowerUp** — даёт игроку временный щит на 5 секунд (поглощает 1 удар, визуально — голубое мерцающее кольцо вокруг игрока).
    2. **DoubleFirePowerUp** — удваивает скорострельность игрока на 8 секунд (уменьшает cooldown вдвое).
    3. **HealPowerUp** — восстанавливает 1 HP игроку (максимум 3 HP, не превышать).
    Усилители падают сверху вниз со скоростью 150 px/s. Отрисовка — цветные квадраты 40x40 с иконкой внутри (щит — голубой, огонь — оранжевый, аптечка — зелёный). В `WaveDirector.js` добавить таймер спавна усилителей: каждые **12 секунд** с разбросом ±3 сек (`12 + (Math.random() * 6 - 3)`) спавнить один случайный усилитель в случайной позиции по X. В `CollisionManager.js` добавить проверку коллизии `player` против `powerup` — при подборе применять эффект и удалять усилитель. Если Y > 2000 — `markedForDeletion = true`.
  - **Ограничения:** Эффекты DoubleFirePowerUp и ShieldPowerUp не стакаются — повторный подбор сбрасывает таймер. Не более 3 активных усилителей на экране одновременно (WaveDirector не спавнит новые, пока старые не подобраны/улетели).
  - **Критерии приемки [QA]:** Запустить игру. Через ~12 секунд должен появиться усилитель сверху. Подобрать его игроком — эффект должен применяться (визуально заметно для щита и двойного огня). HP не должен превышать 3 после подбора аптечки при полном здоровье.

- [x] {{TASK:5.2}} Босс конца уровня. `[Role: Game Gameplay Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/Boss.js`. Обновить `WaveDirector.js` и `CollisionManager.js`.
  - **Как сделать:** Создать класс `Boss` (`type = 'enemy'`, `hp = 30`). Босс появляется через 30 секунд после старта уровня. При спавне босса `WaveDirector` прекращает спавн обычных врагов. Босс медленно входит сверху (Y от -200 до 300) и останавливается. Визуально — крупный корабль (width=200, height=160), нарисованный процедурно средствами Canvas API (тёмно-красный корпус с деталями). **Вооружение босса (3 орудия):**
    1. **Прямой залп** — каждые 1.5 сек стреляет 3 снаряда вниз веером (тип `boss_projectile`).
    2. **Круговой обстрел** — каждые 4 сек выпускает 8 снарядов по кругу (360°).
    3. **Прицельная ракета** — каждые 6 сек один медленный снаряд, летящий в сторону текущей позиции игрока (направление задаётся при выстреле, не самонаводящаяся).
    Снаряды босса (`type = 'boss_projectile'`) наносят урон игроку при коллизии. В `CollisionManager.js` добавить проверку `player` против `boss_projectile` (урон 1 HP). Снаряды игрока попадают в босса. При HP босса <= 0 — большой взрыв частиц (вызов ParticleSystem), босс удаляется. После гибели босса можно показать простую надпись «VICTORY» на канвасе.
  - **Ограничения:** Босс не двигается по Y после остановки, но плавно перемещается по X (синусоида с малой амплитудой ~80px). Снаряды босса за пределами экрана (Y > 2000 или Y < -50) получают `markedForDeletion`. Не более 50 активных снарядов босса одновременно.
  - **Критерии приемки [QA]:** Запустить игру и подождать 30 секунд. Босс должен появиться сверху. Босс должен стрелять тремя типами атак. Игрок должен получать урон от снарядов босса. При уничтожении босса — взрыв частиц и надпись «VICTORY».
