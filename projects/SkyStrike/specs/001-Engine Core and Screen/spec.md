## Tasks
- [x] {{TASK:1.1}} Настройка верстки (Letterboxing) и HTML/CSS основы. `[Role: UI/UX Designer]`
  **ПОДРОБНОСТИ:** - **Что сделать:** Создать `index.html` и `style.css`. - **Как сделать:** Задать черный фон для `body`. Создать контейнер для `canvas`, который сохраняет пропорции 9:16 (максимум по высоте экрана) и центрируется по горизонтали. Использовать CSS (например, `aspect-ratio: 9/16; max-height: 100vh; margin: 0 auto; display: block;`). Разместить UI-слой абсолютным позиционированием поверх канваса. - **Ограничения:** Запрещено растягивать Canvas на `window.innerWidth`, если это нарушает пропорции. - **Критерии приемки [QA]:** Открыть страницу в Agent-Browser. При изменении размера окна браузера (широкоформатный и узкий) игровой экран всегда должен оставаться вертикальным прямоугольником по центру.
- [x] {{TASK:1.2}} Ядро движка (Game Loop) и DeltaTime. `[Role: Game Engine Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/Engine.js` и `src/main.js`.
  - **Как сделать:** В `Engine.js` реализовать класс с массивом `entities`. Цикл `requestAnimationFrame`. Расчет `deltaTime` в секундах. Ограничить `deltaTime` максимумом `0.1` сек (защита от ухода вкладки в фон). В `update(dt)` вызывать апдейты сущностей, затем фильтровать тех, у кого `markedForDeletion === true`. В `main.js` инициализировать движок. Логический размер канваса при старте задать 1080x1920.
  - **Ограничения:** Вся отрисовка только через контекст canvas. Не использовать сторонние библиотеки.
  - **Критерии приемки [QA]:** Запустить страницу, в консоли не должно быть ошибок. Вывести в консоль `FPS` один раз через 3 секунды работы для проверки цикла.
