## Tasks
- [x] {{TASK:2.1}} Сущность Игрока и ввод. `[Role: Game Gameplay Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/Player.js`.
  - **Как сделать:** Класс `Player` с `type = 'player'`, `hp = 3`. Обработка событий клавиатуры (WASD/Arrows). Игрок должен плавно перемещаться, учитывая `deltaTime`. Игрок не может вылететь за пределы логического экрана (0..1080 по X, 0..1920 по Y). Отрисовка: процедурный треугольник средствами Canvas API (заглушка самолета).
  - **Ограничения:** Управление должно быть привязано к флажкам кнопок (keys = {up: false...}), а физика применяться в `update()`.
  - **Критерии приемки [QA]:** Нажать стрелку вправо. Игрок должен долететь до правого края и остановиться, не вылетая за рамки канваса.

- [x] {{TASK:2.2}} Снаряды игрока. `[Role: Game Gameplay Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/Projectiles.js`. Обновить `Player.js`.
  - **Как сделать:** Класс `PlayerProjectile` с `type = 'player_projectile'`. При нажатии на Space в `Player.js` спавнить снаряды, летящие вверх (Y уменьшается). Снаряды, чей Y < -50, должны получать `markedForDeletion = true`.
  - **Ограничения:** Ограничить скорострельность (cooldown timer), чтобы не спавнить 60 пуль в секунду.
  - **Критерии приемки [QA]:** Зажать Space, убедиться что пули вылетают с интервалом. Подождать 10 секунд и проверить размер массива `engine.entities` (пули должны удаляться, массив не должен бесконечно расти).
