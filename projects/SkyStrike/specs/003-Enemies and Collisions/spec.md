- [x] {{TASK:3.1}} Базовый враг (Scout) и Wave Director. `[Role: Game Gameplay Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/Enemies.js` и `src/WaveDirector.js`.
  - **Как сделать:** Создать класс врага `Scout` (`type = 'enemy'`, движение вниз по синусоиде). Если Y > 2000 — удалять (`markedForDeletion`). `WaveDirector` (`type = 'director'`, не имеет отрисовки) каждые 3 секунды спавнит 3-х врагов сверху экрана за пределами видимости (Y = -100).
  - **Ограничения:** Инстанцировать WaveDirector в `main.js`.

- [x] {{TASK:3.2}} Менеджер коллизий (Строгий). `[Role: Game Engine Developer]`
  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/CollisionManager.js`.
  - **Как сделать:** Класс с `type = 'system'`, не имеет отрисовки. В `update()` перебирать сущности. Искать по типам: `player_projectile` против `enemy` (проверка пересечения окружностей). При попадании — уменьшать HP врага, удалять снаряд. Если HP врага <= 0, удалять врага. Коллизия `player` против `enemy` (урон игроку). **КРИТИЧНО:** Использовать фильтрацию по полю `type`, чтобы игрок не бил сам себя.
  - **Ограничения:** Код проверки должен быть оптимизирован.
  - **Критерии приемки [QA] - SURVIVAL TEST:** Запустить игру. Ничего не нажимать 5 секунд. Игрок НЕ должен взорваться сам по себе при старте. Дождаться, пока враг врежется в игрока — HP игрока должно уменьшиться. Запустить игру, пострелять по врагам — они должны уничтожаться.
