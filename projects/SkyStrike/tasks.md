## Спринт 3: Враги и Коллизии [Role: Game Engine Developer & QA]

  **ПОДРОБНОСТИ:**
  - **Что сделать:** Создать `src/CollisionManager.js`.
  - **Как сделать:** Класс с `type = 'system'`, не имеет отрисовки. В `update()` перебирать сущности. Искать по типам: `player_projectile` против `enemy` (проверка пересечения окружностей). Коллизия `player` против `enemy` (урон игроку). **КРИТИЧНО:** Использовать фильтрацию по полю `type`, чтобы игрок не бил сам себя.

  - **Критерии приемки [QA] - SURVIVAL TEST:** Запустить игру. Ничего не нажимать 5 секунд. Игрок НЕ должен взорваться сам по себе при старте. Дождаться, пока враг врежется в игрока — HP игрока должно уменьшиться. Запустить игру, пострелять по врагам — они должны уничтожаться.
