# GEMINI.md — Rules for ModernToDo

## ⚡ СТРОГИЕ ПРАВИЛА
1. **Theming:** ЗАПРЕЩАЕТСЯ хардкодить цвета. Используй CSS-переменные для всех фонов, текстов и теней. Светлая и темная темы должны переключаться плавно.
2. **Architecture:** Логика JS должна быть отделена от DOM-структуры (модульный подход).
3. **Verification:** Перед сдачей UI-задачи обязательно проверить рендер в браузере (Agent-Browser). Проверить работу LocalStorage (данные не должны пропадать при перезагрузке).
4. **No Legacy:** Используй современные методы JS (Optional chaining, Nullish coalescing, template literals).
