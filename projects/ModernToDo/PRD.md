# PRD — Modern HTML ToDoApp

## 1. Vision
Создать эстетичное, современное ToDo-приложение в виде одной (или нескольких) HTML-страниц без тяжелых фреймворков, но с использованием передовых техник верстки и JS.

## 2. Key Features
- **Modern Design:** Стилистика Glassmorphism (эффект матового стекла).
- **Dark/Light Mode:** Автоматическое переключение и ручной тумблер.
- **Persistence:** Сохранение задач в LocalStorage.
- **Animations:** Плавное появление и удаление задач (Web Animations API или CSS Transitions).
- **Responsive:** Идеальное отображение на мобильных устройствах.

## 3. Tech Stack
- HTML5, Modern CSS (Variables, Flexbox, Grid).
- Tailwind CSS (via Play CDN for prototyping).
- Vanilla JavaScript (ES6+).
- Lucide Icons (via CDN).
